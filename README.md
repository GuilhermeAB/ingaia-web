## Contribuições
```sh
# Instalar dependencias
yarn install
```

```sh
# Executar aplicação em modo de desenvolvimento
yarn serve
```

```sh
# Build produção
yarn build
```

### Testes unitários
```
yarn test:unit
```

### Testes end-to-end
```
yarn test:e2e
```
